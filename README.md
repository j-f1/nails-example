> Example [Nails](https://github.com/ArtOfCode-/nails) app ([Demo](https://nails-example.glitch.me/status))

## Install
1. Install [Node](https://nodejs.org) and [npm](https://npmjs.org)
1. `npm install`

## Usage
1. `npm start`
1. Visit http://localhost:8080/status
